# DOM?

- 문서 객체 모델
- Document Object Model
- HTML 문서의 모든 요소의 접근 하는 방법을 정의한 API
- 노드구조로, 트리 형태로 표현함
- HTML DOM, XML DOM, Core DOM으로 구분



## DOM 의 트리구조

```html
<Students>
    <Student>
        <name>1student</name>
        <age>18</age>
    </Student>
    <Student>
        <name>2student</name>
        <age>17</age>
    </Student>
    <Student>
        <name>3student</name>
        <age>16</age>
    </Student>
</Students>
```
